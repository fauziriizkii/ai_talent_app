# AI Talent Success Dashboard

This project was developed as part of the 2025 Data Analyst Case Study.
It transforms complex employee performance data into actionable, explainable insights.

## Features
- Dynamic input: Role name, level, purpose, and benchmark employee IDs
- Automatic benchmark recalculation and SuccessScore computation
- AI-generated role profile and interactive visualizations
- Live dashboard deployment via Streamlit Cloud

## Tech Stack
Python | Streamlit | Pandas | Plotly | Matplotlib | Seaborn
